This guide will walk you through deploying your first application with Globe.dev in just a few steps. Whether you're building a Dart backend or a Flutter web app, Globe.dev makes deployment simple.

## What You'll Need:

- [**Dart/Flutter SDK**](https://dart.dev/get-dart) installed.
- **Globe CLI** set up — run `dart pub global activate globe` if you haven’t already.
- A [**Globe.dev account**](https://globe.dev/).

## What You'll Learn:

In this guide, you'll:

- Deploy a simple Dart backend using **Shelf**.
- Deploy a basic **Flutter web app**.
